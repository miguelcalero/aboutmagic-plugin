== Description ==

AboutMagic let you generate page with profiles from about.me. This plugin use cache system so you can access to about.me API and save results for unlimited use.

== Installation ==

This section describes how to install the plugin and get it working.

0. You need have active curl_init function in your server
1. Unpack file `aboutmagic.zip` in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= It not work =

Are you sure that you have curl_init active?
